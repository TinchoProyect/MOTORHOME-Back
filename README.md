# MOTORHOME-Back (Chasis LAMDA)

Backend del Sistema de Gestión de Proveedores (v2.0).

## Estructura
- /src: Código fuente del servidor (Express).
- /public: Archivos estáticos.

## Instalación
1. `npm install`
2. Configurar `.env` basado en `.env.example`.
3. Colocar `service-account.json` en la raíz.

## Ejecución
- Dev: `npm run api` (Puerto 5555)
